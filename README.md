# Mina Webpacker

Simple gem to add Mina tasks for compiling assets using the webpacker gem.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "mina-webpacker"
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mina-webpacker

## Tasks

 - webpacker:compile
